PASO 1

Arquitectura

DirectOS tiene dos componentes principales que trabajan juntos:

  • Frontend (index.html)
    Single Page Application con Pipeline Builder, Prompt Builder, y Dashboard.
  • Backend (FastAPI)
    API REST en puerto 8000. Ejecuta pipelines, Claude CLI, y gestiona datos.

    ▼

PASO 2

Arrancar DirectOS

Sigue estos pasos para poner DirectOS en marcha: 

# 1. Crear entorno virtual
python3 -m venv venv
source venv/bin/activate

# 2. Instalar dependencias
pip install fastapi uvicorn

# 3. Arrancar
./start.sh
Backend corriendo en http://localhost:8000
Terminal
$ ./start.sh
INFO: Uvicorn running on http://0.0.0.0:8000
INFO: Application startup complete.
$ curl localhost:8000/api/health
{"status":"healthy","version":"10.9.0"}

PASO 7

Claude CLI Integration

La característica estrella de DirectOS v10.3. Ejecuta Claude usando tu suscripción Pro/Max. Sin coste de API.

Usa tu suscripción existente
Sin límites de tokens por pago
Mismo modelo que Claude.ai
Requisitos
  • • Claude Code CLI instalado
  • • Autenticado con claude login
  • • Suscripción Pro o Max activa
Cómo funciona internamente:
# DirectOS ejecuta esto:
claude -p "tu prompt" --output-format json

# Y parsea la respuesta JSON
{
  "result": "Respuesta de Claude...",
  "model": "claude-sonnet-4"
}
Probar endpoint:
curl -X POST localhost:8000/api/claude/ask -H "Content-Type: application/json" -d '{"prompt": "Hola"}'

PASO 11

Prompt Builder Pro

Constructor visual para crear slash commands de Claude Code con el patrón de 6 secciones.

1 Metadata description, allowed-tools
2 Variables $ARG, dinámicas
3 Workflow Pasos S-TIER
4 Instructions Reglas y guías
5 Report Formato de salida
Ejemplo: /code-review
---
description: Code review de calidad
allowed-tools: Read, Grep, Glob
argument-hint: [archivo o directorio]
---

# Workflow
1. Leer el archivo o directorio
2. Analizar calidad del código
3. Identificar problemas
4. Sugerir mejoras

# Instructions
- Buscar code smells
- Verificar naming conventions
- Revisar complejidad ciclomática

# Report
Formato estructurado con secciones
